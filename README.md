# 🧬 Disease Biomarker Identification Project

## 📌 Overview

This project focuses on identifying potential disease biomarkers using gene expression data. It compares healthy vs disease samples to find important genes.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* SciPy

## 📊 Features

* Data cleaning
* Fold change calculation
* Biomarker detection
* Volcano plot visualization
